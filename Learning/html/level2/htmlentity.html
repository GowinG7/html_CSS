Html entities are the text(string) that starts with the ampersand and end with the semi-colon.
            - used to display the html reserved word and used to add spaces as we Watching
            - such special text or symbol can't be found on the standard keyboard so html entities are used

Some html entities

-> stars ☆
->copyright symbol ©
->registered symbol ®
-> less than <
-> greater than >
-> quotation "
★ @
